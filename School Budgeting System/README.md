## School Budgeting System

This is a Management API used to track the school's income and expenses.

## Use Cases

### CRUD
- Create resources
- Read resources with filtering options
- Update resources
- Delete resources

### Reports
- Summary of all income:
  - Student fees
  - Student registration
  - Fines
  - Book sales
- Summary of all expenses:
  - Staff salaries
  - School supplies (books, furniture)
  - Maintenance (repairs, building costs)
  - Equipment purchases
  - Utilities (electricity, water, internet, gas)
- Summary of unpaid income
- Summary of unpaid expenses
- Summary of paid income
- Summary of paid expenses

### Generated Files
- PDF and Excel files for each report

### Dashboard
- No dashboard needed

### Analytics
- No analytics needed
